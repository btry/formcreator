Class PluginFormcreatorActorField

Actors field is a field which accepts several users. Those users may be users from the itemtype User or email addresses. Email addresses allows to add actors who don't have an account in GLPI.

PluginFormcreatorField implements PluginFormcreatorFieldInterface
